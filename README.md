# Dashboard Features Checklist

A structured list of features for tracking daily and monthly spending, insights, and overall financial status.

---

## Day & Special Occasions
- [ ] Display the current day (e.g., Monday, January 29)
- [ ] Fetch information from my personal calendar for special days
- [ ] Show a description of any special day

## Spending Summaries
- [ ] Summary of today's expenses
- [ ] Summary of this week's expenses
- [ ] Summary of this month's expenses

## Daily Average Comparison
- [ ] Display a box comparing today’s spending vs daily average
    - [ ] Calculate percentage difference
    - [ ] Show message like: "+23% more spending than average"

## Expense List
- [ ] Display a detailed list of today’s expenses

## Insights
- [ ] Highlight the category with the highest spending today
    - [ ] Display a large card with icon and total amount
- [ ] Track streaks or habits
    - [ ] Example: "This is your 4th consecutive day without Take Out"

## Charts & Visualizations
- [ ] Bar chart showing spending over the last 7 days

## Monthly Status
- [ ] Show total spent vs total received
- [ ] Show remaining days in the month
- [ ] Calculate daily spending limit to stay on track

## Category Breakdown (if `categoryType` is available)
- [ ] Display 3 small percentage bars for top categories this month
